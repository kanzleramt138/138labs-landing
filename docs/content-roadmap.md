# Content Roadmap

## V1

Expand the current landing page without changing the technical stack.

### Must-have

- sharper hero copy
- services section
- use-case section
- delivery model section
- stronger CTA area

### Nice-to-have

- short founder note
- selected tooling / stack row
- availability / response expectation

## Copy priorities

The copy should answer these questions in order:

1. What does 138Labs build?
2. What kind of company or person is it for?
3. What business problem gets solved?
4. Why work with you instead of a generic agency?
5. How should someone reach out?
